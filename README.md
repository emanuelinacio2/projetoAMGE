# Documentação ADS

Site em Next.js para documentar Projetos e Trabalhos das aulas de ADS.

## Rotas principais

- / (home)
- /escopo
- /sprints
- /processos
- /trabalhos
- /projetos

## Como rodar

```bash
npm run dev
```

Abra o navegador em http://localhost:3000.

## Onde editar

- Home: app/page.tsx
- Seções: app/escopo, app/sprints, app/processos, app/trabalhos, app/projetos
- Estilo global: app/globals.css
